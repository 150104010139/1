 标     题 :   Internet 上 的 常见 服务器

 所谓 服务器 （ Server ) 就是 提供 各类 信息 服务 的 计算机 。 目前 Internet 中 的 服务器   
 常见 的 有 如下 这些 ：   
 　 　 ・ Archie   server : 文件 搜索 服务器   
 　 　 ・ BBS   server : 电子 布告栏 服务器   
 　 　 ・ DNS   server : 域名 服务器   
 　 　 ・ FTP   server : 文件传输 服务器   
 　 　 ・ Gopher   server : Gopher 信息 查询 系统 服务器   
 　 　 ・ News   server : 网络 论坛 服务器   
 　 　 ・ POP   server : 电子邮件 接收 服务器   
 　 　 ・ PPP / SLIP   server : PPP / SLIP 拨 接线 服务器   
 　 　 ・ SMTP   server : 电子邮件发送 服务器   
 　 　 ・ Terminal   server : 以 终端 模式 连线 的 服务器   
 　 　 ・ WWW   server : 全球 信息网 服务器   
 　 　 ・ WAIS   server : 是 一种 从 网络 中 检索 数据库 内容 的 工具 系统 ， 被 称为 数据库   
 的 数据库 。
